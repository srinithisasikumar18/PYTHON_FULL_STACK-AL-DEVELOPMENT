1.MATH:
Basic
	ceil(x) → Round up
	floor(x) → Round down
	fabs(x) → Absolute value	
	factorial(x) → Factorial of a number
	gcd(a, b) → Greatest common divisor
	Power & Logarithm
	sqrt(x) → Square root
	pow(x, y) → x to the power y
	log(x) → Natural logarithm
	log10(x) → Base-10 log
	exp(x) → eˣ
Trigonometry
	sin(x), cos(x), tan(x)
	asin(x), acos(x), atan(x)
	degrees(x) → Convert radians → degrees
	radians(x) → Convert degrees → radians
Useful Extras
	hypot(x, y) → √(x² + y²) (distance formula)
	trunc(x) → Truncate decimal (remove fraction)
Constants (used often)
	math.pi → 3.14159…
	math.e → 2.718…
	math.inf → Infinity
	math.nan → Not a Number

2.RANDOM:
random.random() → Random float (0–1)
random.randint(a, b) → Random integer
random.choice(seq) → Random element from list
random.sample(seq, k) → Random multiple elements
random.shuffle(seq) → Shuffle a list
random.uniform(a, b) → Random float in range

3.DATETIME:
datetime.date.today() → current date
datetime.datetime.now() → current date & time
strftime() → formatting
strptime() → parsing string into date
timedelta → date/time
